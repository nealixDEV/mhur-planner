T.U.N.I.N.G. System Info

■ When you acquire an item to equip in a Roll, upgrade the "costume" with the T.U.N.I.N.G. System and strengthen your character!

• Note: These are subject to change at any point. The order of these quirk sets is not guaranteed to happen in this order specifically.

The T.U.N.I.N.G. System can be used for the Super Cute Babies invented by me, Mei Hatsume.

■ About the T.U.N.I.N.G. System

The T.U.N.I.N.G. System is a system which upgrades "costumes" that can turn the tide of battles!

■ Going through the T.U.N.I.N.G. System

Using Character Data collected from battles, you can set Character Memories in a Slot!

■ Slot Rules Part 1

• Note: It is necessary to set separate Character Memories for each "costume" slot.

▸ Tuning Slot

Character Memories have various Tuning Skill effects, and by setting them you can improve your combat skills.

▸ Special Tuning Slot

If all Tuning Slots are filled, you will be able to set them in a Special Tuning Slot.

Set them in a Special Tuning Slot and you will be able to use a Special Tuning Skill.

■ Slot Unlock Key

Locked Tuning Slots can be unlocked using Slot Unlock Keys!

• Note that the number of Slot Unlock Keys needed will differ depending on the "costume" rarity.

You can get Slot Unlock Keys from Event rewards or from the Shop.

■ Tuning Slot Enhancement Part

Slots have levels, and by using Tuning Slot Enhancement Parts, you can level up and further increase your abilities!

In addition, when reaching the Max Level, a {Level-Max-Icon} will appear.

• Note: The number of Tuning Slot Enhancement Parts needed will differ depending on the "costume" rarity or Slot type.

■ Special Tuning Slot Enhancement Part

Use Special Tuning Slot Enhancement Parts to level up Special Tuning Slots.

■ About MAX LVL

Unlocking the MAX LVL Slot will increase the Max Level for the Special Tuning Slot.

■ T.U.N.I.N.G. System

There are rumors that the Villains are also developing something similar, but my babies won't give up! Everyone, please do your best by using the T.U.N.I.N.G. System!
